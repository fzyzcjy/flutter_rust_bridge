# flutter_rust_bridge Rust runtime

Rust runtime package for flutter_rust_bridge.

This repository is intended to be consumed from Cargo as:

```toml
flutter_rust_bridge = { git = "https://github.com/kumsumit/flutter_rust_bridge_rust.git", package = "flutter_rust_bridge" }
```

The procedural macro crate `flutter_rust_bridge_macros` is included in this repository as `frb_macros`.
