# FRB Lint

## Quick Reference

| Command | Description |
|---------|-------------|
| `./frb_internal lint --fix` | Run all lints and auto-fix (recommended) |
| `./frb_internal lint` | Run all lints (Dart + Rust) |
| `./frb_internal lint-dart` | Dart only |
| `./frb_internal lint-rust` | Rust only |
