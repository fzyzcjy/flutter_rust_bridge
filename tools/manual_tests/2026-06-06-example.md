# Manual Test Example

## Purpose

This example demonstrates the expected structure of a manual regression test report for flutter_rust_bridge. It is not tied to a real bug and should be copied or adapted for future cases where CI cannot realistically reproduce the user's report.

## Why Manual

Use a manual report when the scenario depends on an external device, local host configuration, credentials, interactive UI behavior, flaky infrastructure, or another condition that cannot be encoded as a reliable CI job yet. If a narrowed CI job can reproduce the bad behavior, use an intentional red CI reproduction PR instead of this fallback.
