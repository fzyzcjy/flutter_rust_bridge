# flutter_rust_bridge_hooks

Build hook support for flutter_rust_bridge Native Assets integration.

This package is intended to be used by generated flutter_rust_bridge projects and integration templates.

## Usage

Create a `hook/build.dart` file in a Dart or Flutter package that contains a flutter_rust_bridge Rust crate:

```dart
import 'package:flutter_rust_bridge_hooks/flutter_rust_bridge_hooks.dart';

void main(List<String> args) async {
  await build(args, const FlutterRustBridgeNativeAssetsBuilder());
}
```

By default, the builder expects the Rust crate at `rust/` and routes the generated native library to `src/rust/frb_generated.io.dart`.

The defaults match projects generated by flutter_rust_bridge. Projects with a custom layout can pass `cratePath`, `assetName`, Cargo `features`, and `extraCargoBuildArgs` to `FlutterRustBridgeNativeAssetsBuilder`.
